Repository management

This is the information about adding and editing topics in the repository

All repositories are currently located here:

https://github.com/jwidht


All quizzes are classified by the question and answer languages.

The entry point is the file:

https://github.com/jwidht/LBH-Catalogue/blob/master/bylanguage.psv.txt

This file contains the map of available resource groups to their location.

It is possible and easy to collaborate to create quizzes, accessible to everybody. Just modify the files in the github repository online, and the changes will be available to everybody.
